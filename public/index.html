API (Aplication Programming Interface) backend dasturchilar tamonidan dasturlanadi!

API, dasturlar o'rtasidagi ma'lumot almashishni osonlashtiradi. API orqali dastur, tizim yoki xizmatlar o'zaro bog'lanadi va ma'lumot almashinuvi yoki amalni bajarish uchun talab etilgan funksiyalarni chaqirishi mumkin. API orqali dastur to'g'ridan-to'g'ri boshqa dastur yoki xizmatdan ma'lumot olish, yozish, o'zgartirish va boshqa amallarni amalga oshirishi mumkin..
